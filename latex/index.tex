\hypertarget{index_autotoc_md3}{}\doxysubsection{R-\/\+TYPE}\label{index_autotoc_md3}
This project is part of the Advanced C++ knowledge unit and is focused on networked video game development. You will be tasked with implementing a server and a graphical client for the iconic video game {\itshape R-\/\+Type}, using a game engine designed by you.\hypertarget{index_autotoc_md4}{}\doxysubsection{Objective}\label{index_autotoc_md4}
The goal is to develop a networked version of {\itshape R-\/\+Type}, where players can join forces to fight the evil Bydos. The game will feature a server-\/client architecture, where the server manages the game state and the client displays the game and accepts player input.\hypertarget{index_autotoc_md5}{}\doxysubsection{Game Overview}\label{index_autotoc_md5}
{\itshape R-\/\+Type} is a horizontal shoot \textquotesingle{}em up (shmup) game that became immensely popular in the 1990s. It involves players controlling a spaceship, navigating through levels, and defeating enemy monsters called Bydos. Although {\itshape R-\/\+Type} is not the first game of its genre, it had a significant influence on the shmup genre, leading to several ports and remakes. In this project, you will recreate {\itshape R-\/\+Type} with the added complexity of networked multiplayer and enhanced game engine architecture.

By the end of this project, the game will not only be a fun multiplayer experience but also demonstrate your ability to build a game engine with scalable and maintainable architecture.\hypertarget{index_autotoc_md6}{}\doxysubsection{Project Details}\label{index_autotoc_md6}
